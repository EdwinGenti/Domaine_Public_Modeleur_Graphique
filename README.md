<h1>Extraction du domaine public disponible à Avignon - Par le modeleur graphique de QGIS </h1>


<h2>Description</h2>
Le but du projet est d'extraire les parcelles du domaine public disponible d'une commune. (Les couches doivent être dans le bon SCR métrique !). En somme, le modèle va récupérer les géométries de la commune qui ne se superposent pas avec : Le cadastre, les bâtiments, les routes, une zone tampon de 20 mètres autour des routes, les terrains où la pente est supérieure à 15 degrés ainsi que les surfaces hydrographiques.
<br>
<br>


Ce programme peut être utile aux collectivités locales afin d'obtenir les parcelles publiques constructibles disponibles, avec les différents critères cités au-dessus.
<br>
Le programme s'inspire du schéma réalisé par LECOURT Thibault, mon professeur de l'université d'Avignon.
<br />



<h2>Environnement de travail et logiciel utilisés</h2>
